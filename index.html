How to get and hold LUMA

LUMA is not listed on every exchange. It mainly lives on decentralised exchanges (DEX) on BNB Smart Chain, and is also available on the MEXC platform.

Always double-check the contract address. If an address is different, it is not the same token.

  • 1. Install a BSC wallet MetaMask, Trust Wallet or any wallet compatible with BNB Smart Chain.
  • 2. Add the LUMA token Use the contract address 0xEA6C0908734D7cADc10C01b2e4E9d8e1664EECc9 and 18 decimals.
