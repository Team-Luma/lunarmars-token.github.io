What is Lunar&Mars (LUMA)?

LUMA is a community-driven token representing the technological, financial and human transition between a sustainable lunar base and the first steps toward a Martian settlement.

The idea behind LUMA is simple: create a clear symbol for the path “Moon → Mars”. Instead of promising returns, the project focuses on storytelling, education and experimentation around space-themed crypto assets.

LUMA can be used as a badge for supporters of human exploration beyond the Moon, a tool for creative projects, or a building block for experiments such as games, NFTs, or community incentives.

The contract is based on OpenZeppelin BEP-20 primitives and deploys on BNB Smart Chain for low fees and easy integration with existing wallets and DeFi tools.

Core values Transparency, curiosity, respect for risk, and a long-term view of human exploration.
Use cases Community rewards, personal collections, experimental dApps, and space-themed projects.
Not a security LUMA does not represent equity, debt, or revenue rights. It is a conceptual and symbolic token.
Token mechanics

A simple smart contract, focused on clarity, with mint and burn functions and open burning for all holders.

Network & standard BNB Smart Chain (BSC) using the BEP-20 standard, compatible with most wallets and DeFi platforms.

OpenZeppelin base The contract uses audited OpenZeppelin components for transfers, allowances and standard BEP-20 logic.
How to get LUMA

LUMA lives on BNB Smart Chain. You can store it in any wallet compatible with BEP-20 tokens.
